Welcome to
PHP TestFest

Improving code coverage of the PHP language

PHP TestFest aims to improve the code coverage of the test suite for the PHP programming language. Groups from around the world are encouraged to participate from September through December, meeting physically or virtually, learning how to write and contribute phpt tests. We will award prizes (to be announced) to participating individuals and groups in early January 2018.
